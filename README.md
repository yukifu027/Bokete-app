# アプリ名
Bokete

# 概要
既存のお笑いWebサービス「bokete」のスマホアプリ版です。表示される写真に対して、ボケるコメントをつけ、写真フォルダに保存したり、SNSでシェアすることができます。

# 制作期間
3日間

# 制作背景(意図)

APIとJSON解析、スクリーンショットを撮影する方法を学習するため。
